## ✨ Fitur Utama

- 🚀 **Performa Tinggi:** Menggunakan Next.js App Router.
- 📱 **Desain Responsif:** Optimal di semua perangkat.
- 🔒 **Integrasi Web3:** Siap untuk koneksi wallet (MetaMask) dan interaksi smart contract.
- 🛠 **Type-Safe:** Dikembangkan dengan TypeScript.

## 🚀 Memulai (Local Development)

### 1. Clone Repositori
\`\`\`bash
git clone https://github.com/mrxlolcat/MRxLOLCAT_NFT.git
cd MRxLOLCAT_NFT
\`\`\`

### 2. Instal Dependensi
\`\`\`bash
npm install
\`\`\`

### 3. Setup Environment Variables
Salin file \`.env.example\` menjadi \`.env.local\` dan isi nilai variabelnya:
\`\`\`bash
cp .env.example .env.local
\`\`\`

### 4. Jalankan Server
\`\`\`bash
npm run dev
\`\`\`

---

## 🚀 Deployment (Vercel)

1. Push kode Anda ke GitHub.
2. Hubungkan ke Vercel.
3. Tambahkan Environment Variables:
   - \`NEXT_PUBLIC_TEMPLATE_CLIENT_ID\`
   - \`NEXT_PUBLIC_NFT_CONTRACT_ADDRESS\`
   - \`NEXT_PUBLIC_CHAIN_ID\`
